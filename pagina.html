Estamos haciendo GIT

Git status // muestra el estado del archivo
Git config // modifica la configuracion
Git commit // se envia a la base de datos
Git add // se sube al cache
Git rm // se elimina del cache --force --cached
Git reset // volvemos en el tiempo -- soft y -- reset
Git checkout // sirve para volver a la version anterior todo un proyecto.
Git push // Envia al repositorio remoto
Git pull // copia del repositorio remoto
