Estamos haciendo GIT y tambien GITHUB

Git status // muestra el estado del archivo
Git config // modifica la configuracion
Git commit // se envia a la base de datos
Git add // se sube al cache
Git rm // se elimina del cache --force --cached
Git reset // volvemos en el tiempo -- soft y -- reset
Git diff // hace una comparacion con dos versiones se usa como parametro las versiones a comparar
Git checkout // sirve para volver a la version anterior todo un proyecto.
Git push // Envia al repositorio remoto origin master
Git pull // copia del repositorio remoto
Git branch // se crea una rama paralela
Git merge // une dos ramas
Git pull origin master --allow-unrelated-histories // traemos por primera vez un proyecto
Git remote add origin "url" // anadimos a un sitio remoto nuestro proyecto
Git tag // muestra los tags
Git tag -a "nombre del tag" -m "mensaje" "HASH" // agrega un tag
Git push origin --tags // se envian los tags un tag

Llaves publicas y privadas

ssh-keygen -t rsa -b 4096 -C "email" // creamos nuestras llaves publicas y privadas
ssh-add ~/.ssh/id_rsa // subimos nuestra llave privada
Git remote set-url origin "ssh" // cambia el http a ssh
